# Library — Norfolk AI Voice Agent Factory

This library contains reusable templates, prompt snippets, conversation skills, and tool documentation for building voice agents.

## Sections

- **templates/** — Complete agent templates (orchestrator, support, sales, receptionist)
- **prompts/** — Reusable prompt snippets (greetings, objections, closings, SuperConversations)
- **skills/** — Conversation skill patterns (active listening, lead qualification, appointment booking)
- **tools/** — ElevenLabs tool documentation (transfer, end call)

## Usage

Copy templates as starting points for new agents. Reference prompt snippets in system prompts. Apply skills to improve agent conversational quality.
