# Templates

Agent templates for common use cases. Each template includes a sample system prompt, voice configuration recommendations, and tool setup.

## Available Templates

- **orchestrator/** — Team leader agent that routes calls to specialized sub-agents
- **customer-support/** — Empathetic support agent with issue resolution workflows
- **sales-agent/** — Persuasive closer agent with lead qualification and objection handling
- **receptionist/** — Professional call routing agent handling 100+ transfer destinations
